# right-pad `v1.0`

> **극강의 속도. 제로 블로트. 오직 공백 한 칸.**

문자열 뒤에 공백을 추가하는 세계에서 가장 최적화된 패키지.

## ⚡️ 초고속 시작

```bash
npm i right-pad
```

## 🛠 사용법 (단 한 단어)

```typescript
import r from 'right-pad';

r('안녕'); // '안녕 '
```

## 🚀 성능
- **O(1) 시간 복잡도**
- **의존성 0개**
- **V8 엔진 최적화 문자열 결합**
