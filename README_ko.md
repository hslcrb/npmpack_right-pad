# right-pad

> 문자열 오른쪽에 공백 하나를 추가합니다.

[English](./README.md)

## 설치

```bash
npm i right-pad
```

## 사용법

```ts
import pad from 'right-pad'

pad('안녕') // '안녕 '
```
