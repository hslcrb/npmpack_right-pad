# Contributing

Thank you for considering contributing to `right-pad`!

## How to Contribute

1. Fork the repository.
2. Create your feature branch.
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.

## Development

```bash
npm install
npm run build
npm test
```
